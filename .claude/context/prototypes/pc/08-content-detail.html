React Native 开发实战指南

React Native 已经成为跨平台移动应用开发的首选框架之一。本文将从零开始，带你了解 React Native 开发的各个方面。

React Native 移动开发 教程 实战

前言

React Native 已经成为跨平台移动应用开发的首选框架之一。本文将从零开始，带你了解 React Native 开发的各个方面。

环境搭建

首先，我们需要搭建开发环境：

npm install -g @react-native-community/cli
npx react-native init MyProject

开发工具推荐

  • VS Code - 强大的代码编辑器
  • Flipper - 调试工具
  • Reactotron - 状态管理调试

核心概念

组件生命周期

React Native 组件具有以下主要生命周期方法：

  • componentDidMount() - 组件挂载后调用
  • componentWillUnmount() - 组件卸载前调用
  • useEffect() - Hook 版本的生命周期管理
提示: 使用 Hooks 可以让代码更简洁和易于维护。

实战项目

让我们创建一个简单的待办事项应用：

import React, { useState } from 'react';
import { View, Text, TextInput, TouchableOpacity, FlatList } from 'react-native';

const TodoApp = () => {
  const [todos, setTodos] = useState([]);
  const [inputText, setInputText] = useState('');

  const addTodo = () => {
    if (inputText.trim()) {
      setTodos([...todos, { id: Date.now(), text: inputText, completed: false }]);
      setInputText('');
    }
  };

  return (
    <View style={{ flex: 1, padding: 20 }}>
      <Text style={{ fontSize: 24, fontWeight: 'bold', marginBottom: 20 }}>
        待办事项
      </Text>
      
      <View style={{ flexDirection: 'row', marginBottom: 20 }}>
        <TextInput
          style={{
            flex: 1,
            borderWidth: 1,
            borderColor: '#ccc',
            padding: 10,
            marginRight: 10,
            borderRadius: 5
          }}
          value={inputText}
          onChangeText={setInputText}
          placeholder="添加新任务..."
        />
        <TouchableOpacity
          style={{
            backgroundColor: '#007AFF',
            padding: 10,
            borderRadius: 5,
            justifyContent: 'center'
          }}
          onPress={addTodo}
        >
          <Text style={{ color: 'white', fontWeight: 'bold' }}>添加</Text>
        </TouchableOpacity>
      </View>

      <FlatList
        data={todos}
        keyExtractor={(item) => item.id.toString()}
        renderItem={({ item }) => (
          <View style={{
            padding: 15,
            backgroundColor: '#f9f9f9',
            marginBottom: 5,
            borderRadius: 5
          }}>
            <Text style={{
              fontSize: 16,
              textDecorationLine: item.completed ? 'line-through' : 'none'
            }}>
              {item.text}
            </Text>
          </View>
        )}
      />
    </View>
  );
};

export default TodoApp;

性能优化

渲染优化

  1. 使用 FlatList 代替 ScrollView 处理大量数据
  2. 实现 shouldComponentUpdate 或使用 React.memo
  3. 避免在 render 中创建新对象

内存管理

  • 及时清理定时器和监听器
  • 使用图片压缩和缓存
  • 合理使用状态管理工具

总结

React Native 提供了强大的跨平台开发能力，通过本文的介绍，你应该对 RN 开发有了基本的了解。记住，实践是学习的最佳方式！
